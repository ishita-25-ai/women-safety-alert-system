# 🚨 Women Safety Alert System

A web-based safety system that allows users to send emergency SOS alerts with live location tracking and voice-triggered detection.

---

## 🌟 Features
- 🚨 SOS Emergency Alert Button  
- 📍 Real-time Location Tracking  
- 🎤 Voice Detection (trigger using "help")  
- 🗂️ Alert History Storage  
- ⚡ Fast and simple web interface  

---

## 🧠 Tech Stack
- **Backend:** Python (Flask)  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** SQLite  
- **APIs:** Geolocation API, Web Speech API  

---

## 🔄 How it Works
1. User opens the web app  
2. Location is automatically detected  
3. User presses SOS or says "help"  
4. Alert is sent to backend  
5. Data is stored in database  
6. Alerts can be viewed in history page
